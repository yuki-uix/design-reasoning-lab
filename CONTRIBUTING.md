# Contributing

Contributions should make the research more inspectable, reproducible, or falsifiable.

## Before an experiment

- Link a registered research question.
- State controlled and independent variables.
- Define the procedure and expected evidence before running it.
- Create an evidence directory and manifest using the exact experiment ID.

## After an experiment

- Preserve raw artifacts without editing them.
- Record transformations, redactions, and missing evidence.
- Write facts independently from interpretations.
- State limitations and competing explanations.
- Link every hypothesis or conclusion to supporting and conflicting evidence.

Use the pull request checklist. Do not include secrets, private conversations, personal data, or material you do not have permission to redistribute.
